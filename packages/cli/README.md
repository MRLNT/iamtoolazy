# @iamtoolazy/cli — Fase 2 (next up)

Claude Code / Codex plugin built on `@iamtoolazy/core`:

- Combined skill: terse output + ponytail-style YAGNI ladder that wakes up
  only on coding tasks (adaptive, never a flat tax on every turn)
- `/lazy` (toggle + levels), `/lazy-refine` (PCTF a prompt),
  `/lazy-compress` (shrink CLAUDE.md-style memory files, backup kept),
  `/lazy-stats` (honest ledger), `/lazy-review` (one-line findings)
- The skill is active by default from session start (adaptive, zero cost
  on short prompts); commands are manual overrides, not requirements
