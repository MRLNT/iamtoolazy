# Code of Conduct

Be kind, be honest, be constructive. Concretely:

- Critique code and claims, never people.
- No harassment, discrimination, or personal attacks — anywhere, ever.
- Disagreements are resolved with benchmarks and reasoning, not volume.
- Credit others' work generously (this project exists because caveman and
  ponytail proved the ideas).

Violations: report privately to the maintainer via GitHub. Consequences
scale from a warning to a permanent ban, at maintainer discretion.
